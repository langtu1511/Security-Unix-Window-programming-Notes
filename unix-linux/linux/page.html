--------------------- UBUNTU ------------
* Edit file /etc/network/interfaces
  • auto eth0
  • iface eth0 inet static
  • address 10.0.0.100
  • netmask 255.255.255.0
  • gateway 10.0.0.1  
------------------------ CentOS ---------------------
*  vi /etc/sysconfig/network-scripts/ifcfg-ens33
  • HWADDR=00:0C:29:7D:E4:4A
  • TYPE=Ethernet
  • BOOTPROTO=static / dhcp
  • PEERDNS=no
  • NAME=ens33
  • UUID=3ebfd7fc-1bb9-40fa-b0a6-0b06ddb58984
  • ONBOOT=yes
  • IPADDRS=10.10.1.2
  • NETMASK=255.255.255.0
  • GATEWAY=10.10.1.254
