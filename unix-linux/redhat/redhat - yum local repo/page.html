Note:
1. gpgcheck is optional (If you set gpgcheck=0, there is no need to mention gpgkey)
2. You may find multiple repository files in /etc/yum.repos.d directory, remove the all the repo files except local.repo. This is to ensure that your system do not look for packages from CentOS or Redhat repositories.



Enable the Yum Repository and Install Packages

Our disc is now mounted and now we need to enable the existing repository. This step will actually disable the Internet repositories and enable only the Media repository. For this to work, the disc must be mounted to the directory /media/CentOS.

Enable Media repo and disable all others.
  • #yum --disablerepo=* --enablerepo=LocalRepo

Install packages from the disc to your server using yum as you normally would.
  • #yum install package name
