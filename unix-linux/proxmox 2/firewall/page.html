  • root@sd-61502:/etc/pve/firewall# pve-firewall status
  • ip6tables-save v1.4.14: Cannot initialize: Address family not supported by protocol
  • command '/sbin/ip6tables-save' failed: exit code 1
  • Sollution
  • REF: http://forum.proxmox.com/threads/21015-ip6tables-save-error-in-syslog
  • Solved. It seems that installation has ipv6 disabled.
  • Edit /etc/modprobe.d/local.conf and change line:
  • options ipv6 disabled=0
  • Worked for me.


[+] Enable firewall for cluster, host, node
  • To enable firewall for cluster, add following lines to file /etc/pve/firewall/cluster.fw
    • [OPTIONS]
    • enable: 1
  • To enable firewall for host, add following lines to file /etc/pve/nodes/[host_name]/host.fw
    • [OPTIONS]
    • enable: 1
  • To enable firewall for each VM, add following lines to file /etc/pve/firewall/[VM_ID].fw
    • [OPTIONS]
    • enable: 1
